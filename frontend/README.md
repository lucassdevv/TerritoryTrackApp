# TerritoryTrack Frontend - Angular 19 Client ⚛️

Modern, high-performance UI built with [Angular 19](https://angular.io/) and [Signals](https://angular.dev/guide/signals).

## ✨ UI Features

- **Dashboard**: Real-time status chips and dynamic territory lists.
- **Advanced Filtering**: Custom date range filters (14, 30, and 60 days) to manage territory rotation.
- **Red Theme Styling**: Sleek, modern dark/red design using Vanilla CSS for maximum performance and responsiveness.
- **Standalone Architecture**: Using latest Angular best practices (standalone components, inject function, signals).

## 🚀 Key Modules

- **Territory Dashboard**: Main view to track assignment statuses.
- **Shared Components**: Reusable UI elements (Chips, Layouts, Forms).
- **Services**: API integration with NestJS backend using RxJS and Signals.

## 📦 Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Build for production:
   ```bash
   npm run build
   ```
